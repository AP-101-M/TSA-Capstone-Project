# TSA-Capstone-Project
Stock Market Forecasting using Time Series Analysis predicts stock prices using ARIMA models, volatility analysis, and portfolio optimization. Historical NSE stock data was analyzed in Python and executed virtually on StockGro to compare predicted and actual market performance.

## Stock Market Forecasting using Time Series Analysis

This project analyzes and forecasts stock market prices using ARIMA models, volatility analysis, and portfolio optimization techniques in Python.

### Stocks Analyzed
- TCS
- HDFC Bank
- ITC
- Sun Pharma
- Tata Motors

### Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Statsmodels
- yFinance
- StockGro

### Features
- Time Series Forecasting
- Volatility Analysis
- Portfolio Construction
- Model Comparison
- Prediction vs Actual Analysis
### Dashboard Visualization
  - Tableau dashboard visualization for forecasting comparison
